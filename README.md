## Features
- User authentication (Firebase Auth / JWT)
- Role-based dashboards (Employer / Job Seeker)
- Post & manage jobs
- Apply to jobs and manage applications
- Profile editing with photo upload
- Search, filter, pagination for listings
- Notifications and toasts (success/error)
- Responsive layout (Tailwind CSS)
